## ❌ 如果没有评论

### 检查清单

1. 确认已刷新页面（F5）
2. 确认有AI角色
3. 确认配置了Gemini API Key
4. 查看控制台错误信息

### 常见原因

- API Key未配置或无效
- 网络连接问题
- Gemini API限流
