# 多提供商快速测试

## ⚡ 1分钟快速测试

### Gemini测试（推荐新手）

```
1. 点击右上角"⚙️"
2. 提供商选择：Google Gemini
3. 粘贴API Key（从 https://aistudio.google.com/apikey 获取）
4. 模型选择：gemini-2.5-pro
5. 点击"保存"
6. 返回聊天，发送消息测试
```

**预期：** AI正常回复 ✅

---

## 🔍 验证配置是否生效

### 方法1：查看控制台日志

```
F12 打开控制台 → 发送消息 → 查找：
✅ "🤖 [UniversalAI] 使用提供商: gemini"
✅ "🌟 [Gemini] 使用模型: gemini-2.5-pro"
```

### 方法2：测试不同提供商

```
1. 配置Gemini → 发消息 → 记录回复风格
2. 切换到OpenAI → 发同样的消息 → 对比回复
3. 不同提供商回复风格应该不同
```

## 🚨 常见错误

### 错误1: "请先配置XXX API Key"

**原因：** 选择了提供商但未填写API Key

**解决：**
```
配置面板 → 输入对应的API Key → 保存
```

### 错误2: "API错误: 401 Unauthorized"

**原因：** API Key无效或过期

**解决：**
```
检查API Key是否正确 → 重新生成 → 重新配置
```

### 错误3: "无法解析API响应格式"

**原因：** 自定义API返回格式不兼容

**解决：**
```
确认API兼容OpenAI格式 → 或切换到Gemini/OpenAI
```
