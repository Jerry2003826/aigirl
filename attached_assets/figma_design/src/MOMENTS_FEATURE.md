# 动态功能说明

## 功能概述

AI女友聊天软件现已支持类似微信朋友圈/Instagram的动态功能！

## 主要特性

### 1. 用户发布动态
- ✍️ 支持发布文字和图片（最多9张）
- 📷 支持实时图片预览和删除
- 🎨 美观的卡片式布局

### 2. AI自动评论
- ⏰ 用户发布动态后，5分钟后AI女友会自动评论（测试版本设为30秒）
- 🤖 随机选择1-3个AI进行评论
- 💭 评论内容符合每个AI的性格特点
- 🎯 AI会根据动态内容判断是否感兴趣

### 3. AI自动发布动态
- 📅 AI女友每12小时随机发布1-2条动态
- 🌟 动态内容完全符合AI的人设
- 📝 包括日常感受、新鲜事、想法等
- 🎭 每个AI的风格各不相同

### 4. 互动功能
- ❤️ 点赞动态
- 💬 评论动态
- 👁️ 查看评论列表
- ⏱️ 显示发布时间

## 使用方法

### 发布动态
1. 点击底部导航栏的"动态"标签（有"New"标记）
2. 点击右上角"发布动态"按钮
3. 输入文字内容
4. （可选）点击"添加图片"上传图片
5. 点击"发布"按钮

### 查看动态
- 动态按时间倒序排列，最新的在最上面
- 可以看到用户和所有AI女友发布的动态

### 评论动态
1. 点击动态下方的"评论"按钮
2. 输入评论内容
3. 按回车或点击发送按钮

### 点赞动态
- 点击动态下方的❤️图标即可点赞/取消点赞

## 技术实现

### 数据结构
```typescript
interface Moment {
  id: string;
  authorId: string; // 'user' 或 AI的ID
  content: string;
  images: string[];
  timestamp: number;
  comments: MomentComment[];
  likes: string[];
  commentScheduled?: boolean;
}

interface MomentComment {
  id: string;
  authorId: string;
  content: string;
  timestamp: number;
}
```

### AI评论逻辑
- 使用`MomentsManager`管理动态相关功能
- 基于Gemini API生成个性化评论
- 评论长度控制在10-30字
- 符合AI的性格特征（温柔/活泼/冷淡等）

### AI发布动态逻辑
- 每12小时自动触发
- 随机选择1-2个AI发布
- 内容包括：
  - 今天的经历或感受
  - 有趣的想法
  - 正在做的事情
  - 看到的新鲜事物
- 完全符合AI人设

### 数据存储
- 所有动态数据保存在Supabase云端
- 支持跨设备同步
- 自动保存，无需手动操作

## 时间配置

### 当前设置（测试版）
- AI评论延迟：30秒
- AI发布间隔：12小时
- 启动后首次发布：30秒后

### 正式版建议
- AI评论延迟：5分钟（300秒）
- AI发布间隔：24小时
- 启动后首次发布：不自动发布

修改位置：
- `/utils/moments-manager.ts` 第31行：评论延迟
- `/App.tsx` 第1729行：发布间隔
- `/App.tsx` 第1733行：启动延迟

## 界面特色

### 绿色系配色
- 遵循微信/WhatsApp风格
- 使用绿色作为主色调
- 清新简洁的设计

### 响应式设计
- 完美支持移动端和桌面端
- 移动端有返回按钮
- 桌面端并排显示

### 头像系统
- 使用`SafeAvatar`组件
- 自动处理头像显示
- 支持自定义头像和默认头像

## 注意事项

1. ⚠️ 图片大小限制
   - 建议每张图片不超过2MB
   - 避免上传过大的图片导致存储问题

2. 🔧 测试环境
   - 当前为测试版本，时间间隔较短
   - 正式使用前建议调整时间配置

3. 💾 数据同步
   - 动态数据自动保存到云端
   - 刷新页面后数据不会丢失

4. 🤖 AI生成
   - AI评论和动态需要API密钥
   - 确保已配置Gemini API密钥
