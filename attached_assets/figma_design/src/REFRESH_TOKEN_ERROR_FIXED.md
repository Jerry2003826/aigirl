## 问题描述

错误：`POST https://...supabase.co/auth/v1/token?grant_type=refresh_token 400 (Bad Request)`

这个错误通常发生在：
- Refresh token 过期（默认7天）
- 认证状态不一致
- Token 刷新失败

## 常见问题

### Q: 为什么每次打开应用都要登录？
**A:** 可能是浏览器设置清除了cookie/storage。检查：
- 浏览器是否启用了隐私模式
- 是否设置了自动清除浏览数据
- 是否使用了某些隐私保护扩展

### Q: Token多久会过期？
**A:** 
- Access Token: 1小时
- Refresh Token: 7天（可在Supabase配置）
- 应用会提前10分钟自动刷新

### Q: 如何延长Token有效期？
**A:** 在Supabase控制台：
1. 进入项目设置
2. Authentication → Settings
3. 修改 "JWT Expiry" 和 "Refresh Token Expiry"

### Q: 多设备登录会有问题吗？
**A:** 不会，但注意：
- 在一个设备登出会使其他设备需要重新登录
- 建议在每个设备上单独登录
